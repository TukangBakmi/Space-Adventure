# Space-Adventure
A simple platformer game using Greenfoot IDE

Steps:
1. Download all of this file as a zip file
2. Extract the file into a folder
3. Cut all images in folder "images2" and paste it in folder "images" (it's inside the file you have extracted)
4. You can delete folder "images2" or not
5. Open your greenfoot (if you don't have greenfoot, you can download it from https://www.greenfoot.org/download)
6. In greenfoot, select scenario, and click open
7. Select folder that contains "docs", "images", "sounds", "Acid.class", etc
8. In class world, right click on the "HomePage" and click "new HomePage()"
9. Now you can run it on greenfoot and enjoy the game!
